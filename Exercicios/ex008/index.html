Principais Formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase temos um texto em destaque usando a tag STRONG (semântica)

Italico/Ênfase

Nesta frase temos um texto em Italico usando a tag I (não semântica)

Nesta frase temos um texto em Ênfase usando a tag EM (semântica)

Quando usamos a tag B para negrito ou tag I para Italico, estamos usando forma, não tem sentido. O HTML tem que ser semantica e deixamos a forma para o CSS, então usamos em HTML tag que de sentido aquela parte, como STRONG e EM.

Texto marcado

Podemos criar um texto marcado usando a tag MARK

Para mudar a cor da marcação usamos CSS com o parametro "style" se for pontual usamos dentro do "mark" que quremos, caso seja para todos os mark usamos o style no head, como irei usar aqui... outro texto marcado

Texto grande e texto pequeno

Nesse paragrafo temos um texto grande e outro texto pequeno

A tag big está obsoleta, não deve ser usado pois não tem sentido, podemos deixar o texto grande usando CSS. Ja a tag small tem sentido, pois usamos texto pequeno em algumas situaçoes como em contratos para dar importancia aquela parte.

Texto deletado

POdemos marcar um texto como excluido para indicar que ele deve ser lido mas naão considerado, usamos a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase que ele foi adicionado depois, usamos a tag INS

podemos usar a tag U tbm, porém essa é uma forma (não semântica)

Texto sobrescrito

Um exemplo disso é x3+2, usamos a tag SUP

Texto subscrito

Um exemplo seria a formula da agua H2O, usamos a tag SUB
